RogueNet presents: An Unofficial Roguelike Celebration 2021 Party!

West of the temple has been set up as a festive gathering space, make sure to get the event swag, and some drinks maybe, while you chat with your roguelike friends.

East of the temple you will find the Roguelike Museum, displaying a collection of roguelike games from the 80's. Remember to buy some merch on your way out!

Roguenet features a text-based, ephemeral conversation system. There is no message log, you are meant to be focused on the event while it last (logout and login back if you intend to be away!)

Press [B] to set your badge details, and [S] to toggle sound on/off
